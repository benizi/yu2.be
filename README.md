# yu2.be

Minimal YouTube URL minifier.

## Requirements

Requires [Leiningen](https://github.com/technomancy/leiningen) to manage Clojure dependencies.

Requires [Foreman](https://github.com/ddollar/foreman) for managing runtime processes.

## Installation

```shell
lein deps
```

## Running

```shell
foreman start
```
